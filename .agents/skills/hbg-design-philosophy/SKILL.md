# HBG Design Philosophy

Swiss Modernist design identity for a premium institutional investment tool. Typography: Inter (body), IBM Plex Sans (headings), JetBrains Mono (numbers). Color philosophy: earth-tone warmth (Tuscan Olive Grove default). Motion: framer-motion with subtle transitions. Layout: consistent card widths, progressive disclosure, high data density. Hospitality-native vocabulary throughout.
