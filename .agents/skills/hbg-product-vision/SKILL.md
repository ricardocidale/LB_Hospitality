# HBG Product Vision

"Bloomberg Terminal for boutique hospitality." Five core tenets: deterministic integrity, transparency, professional elegance, hospitality-native vocabulary, progressive disclosure. Workflow principles: minimize navigation, research informs (never dictates), verification builds confidence, scenarios are effortless. Four user roles: admin, partner, checker, investor. White-labeling via user groups with theme/logo cascade.
