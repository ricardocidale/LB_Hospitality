# Settings Architecture

Three configuration surfaces: (1) Management Company — entity identity, revenue model, operating costs, funding; (2) General Settings — property defaults, macro economics, UI preferences; (3) Admin Panel — system administration only. Property Edit for per-property overrides. Overlap prevention rules and decision tree for placing new settings.
