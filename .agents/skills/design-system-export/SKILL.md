# Design System Export

Instructions for replicating the HBG design system in another project. 5-layer architecture: Theme (CSS vars + Tailwind v4), Primitives (shadcn/ui + Radix), Icons (custom SVG), Graphics (KPI grids, animations), Charts (Recharts + Mermaid). Copy index.css, utils.ts, components/ui/, components/icons/, components/graphics/ for full visual identity.
